# 🧾 Store Expense Tracker

A simple and user-friendly web application that helps store owners manage and track daily expenses efficiently.

---

## 📌 Project Overview

The Store Expense Tracker is designed to help small businesses and shop owners record their daily expenses digitally instead of maintaining manual notebooks.

This application allows users to:

- Add expenses
- Edit expense records
- Delete expenses
- View total spending
- Track expenses by category and date

It provides a clean and responsive interface built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- ➕ Add new expense entries
- 📝 Update existing records
- ❌ Delete expenses
- 📅 Track expenses by date
- 🗂 Categorize expenses (Rent, Stock, Electricity, Salary, etc.)
- 📊 Automatically calculate total expenses
- 💾 Data stored using Local Storage

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Browser Local Storage
